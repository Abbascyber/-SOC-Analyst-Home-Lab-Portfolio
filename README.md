# 🛡️ SOC Analyst Home Lab

## 🔍 Overview
This project documents my hands-on SOC lab environment built to simulate real-world security operations including network security, vulnerability assessment, SIEM monitoring, and threat analysis.

---

## 🏗️ Lab Architecture

### ✅ Configured Components

- pfSense → Router & Firewall (network segmentation & port configuration)
- Kali Linux → Installed for penetration testing
- Ubuntu Server → Prepared for SIEM deployment
- Metasploitable → Vulnerable machine for testing

---

### ⚠️ In Progress

- Splunk Enterprise (SIEM setup planned)
- Security Onion (monitoring & threat hunting)

---

### 🚧 Current Limitation

Due to hardware (RAM) constraints, full lab execution is temporarily limited.  
System upgrade is planned to enable full SOC workflows including log ingestion, detection, and incident response.

---

## 🔄 Planned SOC Workflow

- Route traffic through pfSense firewall
- Scan vulnerable machines (Metasploitable) using Kali tools
- Perform vulnerability assessments (Nessus)
- Ingest logs into Splunk SIEM
- Monitor and investigate alerts using Security Onion
